# Hello Dart

This directory contains introductory Dart programming exercises and examples.

## Contents

- Basic syntax and variables
- Functions and control flow
- Object-oriented programming concepts

## Getting Started

1. Install Dart SDK
2. Run exercises with `dart run` or `dart filename.dart`
3. Check official [Dart documentation](https://dart.dev/guides)
